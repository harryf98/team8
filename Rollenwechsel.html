    • Rollenwechsel

    Zuerst überprüfen Sie das System, ob es eine Fehlermeldung anzeigt. Falls die Fehlermeldung anzeigt, dass die Rolle aus ist. Geben Sie zuerst ihre Zugangsdaten ein. Danach fahren Sie das System herunter. Öffen Sie die Lade der Rechnungsrolle. Setzen Sie die neue Rolle ein und anschließend schalten Sie das System wieder hoch. Das System wir automatisch konfiguriert und sie müssen zum Abschluss Ihren Mitarbeiter Code eingeben.
